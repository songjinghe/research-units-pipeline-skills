# Idea Memo Writer — Report Structure

## Expected Sections

1. **Topic**: the research question or goal
2. **Top Ideas** (3 ranked): each with title, thesis, novelty angle, evidence links
3. **Deferred Ideas**: ideas not prioritized, with brief rationale
4. **Trace Provenance**: links to upstream trace artifacts

## Formatting Rules

- Use Markdown headers for sections
- Each idea should have: `## Idea N: [Title]` followed by thesis, novelty, evidence
- Deferred ideas use a compact bullet list
- No placeholder text (`TODO`, `TBD`, etc.)
- JSON payload mirrors the Markdown structure for programmatic consumption
