# Block vs Downgrade

## Block when

- citation keys are unusable or absent
- evidence is title-only
- no snippets are extractable
- comparison structure is too weak to produce grounded A-vs-B cards
- the subsection clearly calls for evaluation anchors but none can be extracted

## Downgrade when

- evidence is mostly abstract-level rather than fulltext-backed
- limitations/failure evidence is too sparse for strong conclusions
- numeric claims exist but protocol context is incomplete

## Verify fields

Use `verify_fields` for non-blocking gaps such as:
- benchmarks/datasets
- metrics / human-eval setup
- budget / tool access / horizon
- baseline / ablation details
- failure modes / known limitations
