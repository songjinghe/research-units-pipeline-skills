## Evaluation protocol

- Evaluation mentions include: RIG, EVLP, WBC, BFMs, BFM, FMRs, ACT, VLA, VLAs, CRT. Zhao2025Synergizing Francis2023Core Francis2021Core Yuan2025Survey
- When comparing results, anchor the paragraph with: task type + metric + constraint (budget, tool access, horizon, or threat model) when stated. Zhao2025Synergizing Francis2023Core
- Prefer head-to-head comparisons only when benchmark/metric are shared; otherwise frame differences as protocol-driven rather than method superiority. Zhao2025Synergizing Francis2023Core
- Avoid underspecified model/baseline naming; if abstracts omit details, state that the baseline is reported but underspecified instead of guessing. Zhao2025Synergizing Francis2023Core
- If a claim relies on a single reported number, pair it with a limitation/caveat from the same evidence so the draft remains conservative. Zhao2025Synergizing Francis2023Core
- If budgets or environments differ across papers, treat cross-paper numeric comparison as fragile and prefer qualitative contrasts aligned to the subsection axes. Zhao2025Synergizing Francis2023Core
