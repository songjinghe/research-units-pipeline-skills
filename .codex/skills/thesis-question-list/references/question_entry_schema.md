# Question Entry Schema

建议每条问题至少包含以下字段：

- `priority`
- `chapters_or_files`
- `problem`
- `why_it_matters`
- `fix_direction`
- `multi_agent_review`
- `acceptance`

## 推荐表格列

| 优先级 | 涉及章节/文件 | 问题 | 为什么是问题 | 修改方向 | 多 agent 讨论 | 验收标准 |
|---|---|---|---|---|---|---|

## 优先级建议

- `P0`：结构 / 主线 / 编译阻断
- `P1`：证据 / 引用 / 关键图表
- `P2`：术语 / 交叉引用 / 局部排版
- `P3`：润色 / 去 AI 味 / 节奏优化
